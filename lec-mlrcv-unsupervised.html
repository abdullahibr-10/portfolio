Unsupervised Learning

Oct 2025 MSc Mobile Robotics, UniBonn

Unsupervised learning discovers structure in unlabelled data — clusters, latent factors, compact representations. In robotics, it underpins codebook generation for visual place recognition, point cloud downsampling, and anomaly detection in sensor streams.

1. k-Means Clustering

k-Means partitions n data points into k clusters by alternating between two steps (Lloyd's algorithm):

  1. Assignment: assign each point xᵢ to its nearest centroid: cᵢ = argminₖ ||xᵢ − μₖ||².
  2. Update: recompute each centroid as the mean of its assigned points: μₖ = (1/|Cₖ|) Σᵢ∈Cₖ xᵢ.

The algorithm monotonically decreases the within-cluster sum of squares J = Σᵢ ||xᵢ − μ_{cᵢ}||² and converges to a local minimum. Because it is sensitive to initialisation, k-Means++ spreads initial centroids by sampling proportionally to squared distance from the nearest chosen centroid — this usually gives faster convergence and better local minima.

2. Choosing k

There is no single correct answer for k. Two practical methods:

  • Elbow method: plot J vs k. The "elbow" — a kink where the rate of decrease sharply slows — indicates a good k. Often subjective.
  • Silhouette score: for each point, s = (b − a) / max(a, b) where a = mean intra-cluster distance, b = mean nearest-cluster distance. s ∈ [−1, 1], higher is better. Average over all points gives a global score to compare different values of k.

3. Gaussian Mixture Models

GMMs model the data as a mixture of K Gaussians. Each cluster k has a mean μₖ, covariance Σₖ, and mixing weight πₖ (with Σπₖ = 1). The data likelihood is:

p(x) = Σₖ πₖ · N(x ; μₖ, Σₖ)

GMMs are fitted with the Expectation-Maximisation (EM) algorithm:

  • E-step: compute soft responsibilities rᵢₖ = πₖ N(xᵢ; μₖ, Σₖ) / Σⱼ πⱼ N(xᵢ; μⱼ, Σⱼ). Each point is assigned to every cluster with a fractional weight.
  • M-step: update parameters using weighted statistics: μₖ = Σᵢ rᵢₖ xᵢ / Nₖ, Σₖ = Σᵢ rᵢₖ (xᵢ−μₖ)(xᵢ−μₖ)ᵀ / Nₖ, πₖ = Nₖ/n where Nₖ = Σᵢ rᵢₖ.

EM monotonically increases the log-likelihood and converges to a local maximum. k-Means is a special case of GMM-EM where covariances are isotropic and equal, and E-step responsibilities are hard (0 or 1).

4. Principal Component Analysis

PCA finds a lower-dimensional linear subspace that maximises variance (equivalently, minimises reconstruction error). The algorithm:

  1. Mean-centre the data: x̃ᵢ = xᵢ − x̄.
  2. Compute the covariance matrix: C = (1/n) X̃ᵀX̃.
  3. Eigendecompose: C = VΛVᵀ. Columns of V are eigenvectors (principal components); Λ contains eigenvalues.
  4. Project onto the top d eigenvectors: z = Vᵈᵀx̃.

The fraction of variance retained is Σᵢ₌₁ᵈ λᵢ / Σᵢ λᵢ. Whitening additionally scales each component by λᵢ⁻¹/², giving uncorrelated unit-variance features — useful as preprocessing for downstream classifiers.

Robotics Applications k-Means builds the visual codebook for Bag-of-Visual-Words (covered next lecture). PCA reduces high-dimensional LiDAR scan features to a compact descriptor. GMMs model multi-modal sensor noise in Bayesian filters. Anomaly detection (e.g., mechanical fault in a joint) fits a GMM to normal operation and flags low-likelihood samples as anomalies.

Unsupervised methods are particularly valuable in robotics because labelled data is expensive to obtain — labelling point clouds or sensor logs requires expert annotation. Learning structure from raw data first, then fine-tuning with limited labels, is an increasingly common paradigm in modern robot perception.
