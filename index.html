Quadcopter Control with Rotatory End Effector
Previous Next

This was my capstone project focusing on developing a robust autonomous quadcopter for contact-based industrial inspection. I designed and implemented an L1-Adaptive Controller in ROS2 to handle uncertainties and external disturbances during flight, ensuring stable contact with surfaces. The entire system was simulated in a high-fidelity Gazebo environment for validation before any hardware implementation.

My Role: Algorithm Developer. I was responsible for the control system architecture, software implementation in C++/Python, and simulation testing.

Technologies Used:
ROS2 Gazebo C++ Python L1-Adaptive Control PX4 Simulink

Pick and Place Tasks for a 4DOF Robot Arm
Previous Next

This was my lab course project focusing on developing a robust pick and place algorithm for a 4DOF robot arm. In this I derived the Inverse and Forward Kinematics of a 4DoF Pincher Arm, calibrated its sensors and performed pick and place tasks with an Intel RealSense camera for perception in a dedicated workspace.

My Role: Calibration/Perception Engineer. I was responsible for the calibration task, integrating the Intel RealSense camera for depth perception and object recognition.

Technologies Used:
MATLAB Python Forward Kinematics Inverse Kinematics
