ML for Robotics & Computer Vision

Lec 01

Regression

Linear regression, normal equations, polynomial regression, and regularization (Ridge/Lasso). The bias-variance tradeoff and cross-validation for model selection.

Classification

Binary and multi-class classification. Logistic regression, k-NN, SVMs with kernel trick. Evaluation metrics: precision, recall, F1, ROC/AUC.

Unsupervised Learning

k-Means clustering and the Lloyd algorithm. Gaussian Mixture Models (GMM) and the EM algorithm. Principal Component Analysis (PCA) for dimensionality reduction.

Read →
Lec 04

Ensemble Learning

Bagging and bootstrap aggregation. Random Forests with feature randomisation. Boosting: AdaBoost and gradient boosting (GBDT). When ensembles help and when they don't.

Read →
Lec 05

ML for Computer Vision Tasks

Classical CV pipeline: HOG features, sliding window detection, bag of visual words. Evaluation with mAP and IoU. The bridge from handcrafted features to learned representations.

Read →

Introduction to CNNs

Motivation for convolutional networks: spatial invariance and parameter sharing. Convolutional layers, pooling, ReLU activations, and the basic LeNet-style architecture.

Read →

Backpropagation through convolutional layers. Weight initialization (Xavier/He). Batch normalization, dropout, and modern optimizers: SGD with momentum, RMSProp, Adam.

Graph SLAM

Lec 01

Camera Parameters

The pinhole camera model, intrinsic matrix K, and the full projection equation. Radial and tangential lens distortion models, and checkerboard-based calibration.

Read →
Lec 02

DLT — Direct Linear Transform

Homography estimation between planes. Setting up the linear system Ah=0 and solving via SVD. Normalization for numerical stability and DLT applications in AR and image stitching.

Read →
Lec 03

Fundamental & Essential Matrix

The epipolar constraint: x'ᵀFx = 0. Fundamental matrix F (7 DOF), essential matrix E for calibrated cameras, and the 8-point algorithm for linear estimation.

Read →
Lec 04

Fundamental & Essential Matrix — Direct Methods

Hartley normalization for the 8-point algorithm. RANSAC-based robust estimation of F. Sampson distance as an algebraic error proxy. Decomposing E into [R|t].

Read →
Lec 05

Epipolar Geometry

Epipoles, epipolar lines, and the epipolar plane. Computing epipolar lines from F. Stereo rectification: warping to align epipolar lines horizontally for efficient dense matching.

Read →
Lec 06

Triangulation

Recovering 3D points from 2D correspondences across calibrated cameras. Linear DLT triangulation, algebraic vs. geometric vs. optimal error, and degenerate configurations.

Read →
Lec 07

Bundle Adjustment

Joint optimization of camera poses and 3D points via reprojection error minimization. Jacobian structure, Gauss-Newton, and the Schur complement trick for sparsity.

Read →
Lec 08

Bundle Adjustment II

Gauge freedom in reconstruction: the 7-DOF ambiguity and how to fix it. Robust loss functions for outlier rejection. Incremental vs. global BA. Ceres Solver and COLMAP in practice.

Read →
Lec 09

Features — Keypoints

Properties of good keypoints: repeatability, distinctiveness, locality. Harris corner detector, FAST, and Difference-of-Gaussians (DoG) for scale-invariant detection.

Read →
Lec 10

Features — Descriptors

SIFT descriptor: gradient orientation histograms over a 4×4 grid with rotation normalization. Binary descriptors: BRIEF and ORB. Matching strategies and Lowe's ratio test.

Read →
Lec 11

ICP — Known Data Association

When correspondences are given: closed-form SVD solution for the rigid transform. Point-to-point vs. point-to-plane ICP variants and their convergence behavior.

Read →
Lec 12

ICP — Unknown Data Association

Full ICP: alternating between nearest-neighbour data association and SVD alignment. Convergence conditions, initialization sensitivity, robust kernels, and the EM interpretation of ICP.

Read →

Humanoid Robotics

Lec 01

Introduction & Perception Basics

History of humanoid robotics (WABOT, ASIMO, Atlas, G1). Morphology, degrees of freedom, and the control hierarchy. Perception stack: cameras, LiDAR, IMU, and force/torque sensors.

Read →
Lec 02

3D World Representations

Point clouds, meshes, signed distance fields (SDFs), and OctoMap for online 3D environment modelling. RGB-D sensors for humanoids: structured light, stereo, and ToF.

Read →
Lec 03

Active Perception

Passive vs. active sensing. Gaze control and foveal vision for humanoids. Information gain for view planning, saccade planning, and active 3D reconstruction strategies.

Read →
Lec 04

Fundamentals of Manipulation

Grasp types (Napier taxonomy). Contact models: point, line, surface. Form closure and force closure conditions. Friction cones and the convex hull criterion for grasp stability.

Read →
Lec 05

Motion Planning for Manipulation

Joint space vs. task space planning. Analytical and numerical IK solvers. Constrained motion planning: equality and inequality constraints. RRT and PRM for manipulation, MoveIt integration.

Read →
Lec 06

Trajectory Generation & Optimization

Path vs. trajectory. Cubic and quintic polynomial trajectories. Minimum-jerk profiles. Via-point blending. Optimal control formulation for time-optimal and energy-optimal motion.

Read →
Lec 07

Grasping & Pushing

Grasp synthesis: contact point selection and antipodal grasp conditions. Pre-grasp manipulation with push primitives. Quasistatic push mechanics and learning-based grasp planning.
