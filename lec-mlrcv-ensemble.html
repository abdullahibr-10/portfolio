Ensemble Learning

Oct 2025 MSc Mobile Robotics, UniBonn

Ensemble methods combine multiple weak learners into a single strong predictor. They are among the most practically effective tools in classical machine learning — Random Forests and Gradient Boosted Trees consistently top structured-data benchmarks and have clear applications in robotics for terrain classification, sensor anomaly detection, and multi-modal fusion.

1. Decision Trees

A decision tree recursively partitions the feature space by axis-aligned splits. At each node, the algorithm selects the feature and threshold that maximally reduces impurity. For classification, the most common impurity measures are:

  • Gini impurity: G = 1 − Σₖ pₖ². Zero for a pure node (all one class), maximum at 0.5 for binary equal-split.
  • Information gain: reduction in Shannon entropy H = −Σₖ pₖ log pₖ after the split.

Deep trees memorise training data (high variance). Pruning via maximum depth or minimum samples per leaf controls this. Even with pruning, single trees are fragile — small data changes can yield very different trees.

2. Bagging

Bootstrap AGGregatING (Bagging) trains T classifiers on independently drawn bootstrap samples (sample n points with replacement from the training set). Final prediction is a majority vote (classification) or average (regression) over all T models. Why it works: individual models have high variance but low correlation. Averaging T uncorrelated predictors reduces variance by factor T while leaving bias unchanged.

3. Random Forests

Random Forests add one more layer of randomisation to bagging: at each node split, only a random subset of m < n features is considered as split candidates. Typically m ≈ √n (classification) or m ≈ n/3 (regression). This decorrelates the trees further — trees cannot all rely on the same dominant feature, so the ensemble is more diverse. The result is a robust, fast, and often surprisingly accurate classifier that requires little tuning beyond the number of trees T.

Feature Importance Random Forests produce a natural feature importance score: for each feature, sum the weighted impurity decrease across all splits on that feature across all trees. This is valuable in robotics for understanding which sensor channels drive a classification decision.

4. AdaBoost

Boosting builds an ensemble sequentially, each new model focusing on examples the previous models got wrong. AdaBoost maintains a weight wᵢ on each training example (initially uniform). At each round t:

  1. Train a weak learner hₜ on the weighted dataset.
  2. Compute weighted error εₜ = Σᵢ wᵢ · 1[hₜ(xᵢ) ≠ yᵢ].
  3. Compute learner weight: αₜ = 0.5 · ln((1 − εₜ) / εₜ). Correct learners (εₜ < 0.5) get positive weight; worse than random gets negative weight.
  4. Update sample weights: increase weight for misclassified examples, decrease for correct ones. Renormalise.

Final prediction: H(x) = sign(Σₜ αₜ hₜ(x)).

5. Gradient Boosting

Gradient Boosted Decision Trees (GBDT) fit each new tree to the pseudo-residuals — the negative gradient of the loss with respect to current model predictions. For MSE loss, pseudo-residuals equal the actual residuals rᵢ = yᵢ − F(xᵢ). A new tree hₜ is fitted to predict these residuals, then added: F_{t}(x) = F_{t-1}(x) + ν · hₜ(x), where ν is a shrinkage factor (learning rate, typically 0.01–0.1). This generalises to any differentiable loss function — log-loss for classification, quantile loss for interval prediction.

XGBoost adds regularisation on tree weights, second-order gradient information (Newton step), and column/row subsampling to GBDT, making it significantly faster and more regularised than naive GBDT.

6. Ensemble Methods in Robotics

Practical applications where ensemble methods shine:

  • Sensor fusion: combine outputs from camera, LiDAR, and IMU classifiers with a Random Forest meta-classifier trained on their confidences.
  • Terrain classification: haptic, visual, and proprioceptive features combined with GBDT to predict slipperiness or slope.
  • Anomaly detection: Isolation Forest (a variant where trees isolate outliers faster) for detecting unusual sensor readings in manufacturing robots.
